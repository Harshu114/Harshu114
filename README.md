# 💫 About Me:
I'm a Data Engineer, AI/ML Engineer, Data Scientist, and Software Creator — not as separate titles, but as one unified way of thinking and building. I understand data at every stage of its life: from raw ingestion and pipeline design, through statistical analysis and feature engineering, all the way to training, evaluating, and deploying machine learning models that actually work in production.<br>On the AI/ML side, I don't just run notebooks — I architect end-to-end systems. I know the math behind the models, the infrastructure needed to serve them, and the product thinking required to make them useful. From classical ML to deep learning, from NLP to computer vision, from fine-tuning LLMs to building RAG pipelines — I operate across the full spectrum.<br>As a software creator, I build things that last. Full-stack development, API design, system architecture, database optimization — I treat software engineering not as a separate skill but as the backbone that makes everything else deployable and scalable.<br>I sit at the intersection of science and engineering — where experimentation meets production, where curiosity meets execution, and where data becomes decisions. I don't just analyze problems; I build the systems that solve them.
